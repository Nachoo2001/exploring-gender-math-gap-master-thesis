# Exploring the Gender Gap in Mathematics — Master's Thesis

Replication code for my master's thesis analysing the gender gap in mathematics among French children, using the **ELFE longitudinal dataset** (Étude Longitudinale Française depuis l'Enfance, INED).

---

## Repository structure

| File / Folder | Description |
|---|---|
| `analysis.R` | Main script — run this to reproduce all figures and regression tables from the thesis |
| `figures/` | Output figures (PDF) generated by `analysis.R` |
| `archive/` | Original exploratory scripts kept for reference, not meant to be run |
| `Mémoire.Rproj` | RStudio project file — open this to set the working directory automatically |

## How to run

1. Open `Mémoire.Rproj` in RStudio
2. Obtain the ELFE data from [INED](https://www.ined.fr/fr/recherche/grandes-enquetes/elfe/) and place the CSV files in a `data/` folder at the project root
3. Run `analysis.R` — figures are saved to `figures/`, LaTeX tables to `tables/`

> **Note:** The ELFE data is restricted access and cannot be shared publicly. You need to request it directly from INED.

## Data

The analysis uses three ELFE survey files:
- `DATA_DEM_1055_RG.csv` — main child development dataset
- `EQR12_VARIABLESOCIODEMO.csv` — constructed sociodemographic variables
- `EQR39_SCOREIDE.csv` — cognitive development scores

## Dependencies

All R packages are loaded automatically via `pacman::p_load()` at the top of `analysis.R`. The main packages used are `fixest`, `ggplot2`, `dplyr`, `MASS`, and `stargazer`.
